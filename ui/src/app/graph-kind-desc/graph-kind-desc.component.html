In Regular graphs all nodes have the same degree. The maximal allowed value for degree is number of nodes minus one (which denotes complete graph).

Another limit of regular graphs is, that product of degree and nodes must be even number, e.g. if you'd need 3-regular graph, you can use only even number of nodes.

In Complete Graph every node is connected (e.g. there is an edge) with every other node in the graph.

In this kind of graph the number of edges is equal to (avg * nodes) / 2.

Please note that in order to make connected graph, the average degree must be at least 2.

This kind of graph, the degree of all nodes is at least equal to Min degree and at most Max degree.

  • Min degree must be non-negative number smaller than nodes - 1.
  • Max degree must be strictly bigger than Min degree and also smaller than nodes - 1.

This kind of graph defines the minimal degree for the graph. So the degrees in resulting graphs will be on interval <min_deg; nodes - 1> .
